Hello Lipstick!

This is a sample wapp that should help you get started.

All wapps are a single page, web based applications that consist of two components: 'foreground data' and 'background data' which serve different purposes.

Wapp Components

Foreground Data

What you see here is a Foreground Data that is running inside the browser's tab (or in the editor's 'preview panel'). It should only contain application UI parts that require user interaction and/or functionality that doesn't need to run constantly.

To create foreground application it is required to add index.html file. The rest is up to you.

Background Data

In the upper right corner of the editor, you can switch to "Background Data" that will load different files that belong to that component of an application.

Read Help to learn more about application components and how they work.

How to code?

In order to create a wapp, use plain HTML and JavaScript or use preferred frameworks and libraries. With the current version we cannot garantee that things will work out of the box, but you can try and write to us if something goes wrong.

We also recommend to read our documentation to understand how things work and find good use for the "helper functions" we have developed for you to ease the process of retrieving and saving Wappsto data.

Navigation

In current release navigation between site pages will not work.

To support navigation, use JavaScript.

Adding an image

There are multiple ways you can load an image

  • Image that was uploaded as a file in Wapp Creator

  • Image from external source (recommended since you will save some space)

  • Images added through CSS will not load since it is not supported in current release.
